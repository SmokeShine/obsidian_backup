# Linear Transformation

- f(alpha. x) = x.f(alpha)
- f(x+y) = f(x) + f(y)

If both checks pass, then it is linear transformation

![[Pasted image 20251201112208.png]]

Apply the first rule and it will fail. For the second, we need f(x = y)

- f(alpha.x + beta.y) = f(alpha.x) + f (beta.y)
	- alpha.f(x)+beta.f(y)
