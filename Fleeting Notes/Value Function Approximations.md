# Value Function Approximations

Given s
1. Return v(s,w)
2. Return Q(s,a,w) for all actions on that state

Given s,a
1. Return Q(s,a,w) for that state action pair

Differentiable
1. Neural Network
2. Linear combination of features

Non Differentiable
1. Decision Tree
2. Nearest neighbours
